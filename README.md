# E-commerce Sales Dashboard
 
Курсовой проект - интерактивный дашборд для анализа данных о продажах интернет-магазина электроники.

**Студентка:** Саидова Марям 
**Курс:** Анализ данных

## Что внутри
📦 ecommerce-dashboard/
├── 🎯 main.py # Сердце дашборда
├── 🗃️ ddl.py # Генератор базы данных
├── 🔌 db.py # Мост к данным
├── 📋 requirements.txt # Инструментарий
├── 📁 queries/ # Мозг анализа
│ ├── 🏗️ 01_create_tables.sql
│ ├── 👁️ 02_create_views.sql
│ └── 🔍 03_queries.sql
├── 📦 source/ # Исходные данные
└── 🛠️ utils/ # Вспомогательные инструменты

🎨 Что внутри дашборда?
📊 1. Ключевые метрики (KPI Dashboard)
💰 Общая выручка с динамикой роста

📦 Количество заказов по статусам

👥 Уникальные клиенты

🏷️ Средний чек

📈 2. Визуализация данных
📅 Динамика продаж по месяцам (линейный график)

🏆 Топ товаров по выручке (столбчатая диаграмма)

🎯 Распределение по категориям (круговая диаграмма)

🔄 Статусы заказов (секторная диаграмма)

🔍 3. Интерактивные фильтры
🎚️ Слайдер для выбора диапазона дат

🔘 Выпадающие списки для фильтрации по категориям

✅ Чекбоксы для выбора статусов заказов

🏙️ Мультивыбор по городам клиентов

📋 4. Raw Data Explorer
👀 Просмотр сырых данных в реальном времени

🔎 Поиск и фильтрация прямо в таблицах

📥 Экспорт данных в CSV

🛠️ Технологический стек

Streamlit	Создание веб-интерфейса	🎈

SQL	Язык запросов к данным	🗃️
